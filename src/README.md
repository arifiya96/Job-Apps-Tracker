# Navigating through the folder structure

### `Components`

This is where the UI components sit. 

AddApplicationModal - The modal responsible for adding a new job application<br>
ColumnData - Specifying the columns for the data grid<br>
ColumnDataStyle - Styling the data grid<br>
TestData - A place to use test data that mimics a backend database<br>
UpdateApplicationModal - The modal responsible for updating a job application

### `Context`

Only one file (ApplicationContext). This is where all the state is handles via the context api.

### `Pages`

Different pages of the app sit here.

Homepage - the homepage of the app (with the user's job applications) when a user logs in.
